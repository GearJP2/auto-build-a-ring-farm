# 🚜 Roblox: Build A Ring Farm (AutoHotkey)

> **Automate your gameplay efficiently.** This repository provides lightweight AutoHotkey (AHK) v2 scripts to automate tedious tasks in Roblox's *Build A Ring*. 

To ensure maximum stability and ease of use, the automation is split into two standalone scripts.

---

## 🛠️ Installation & Setup

Before running the scripts, make sure you have the correct version of AutoHotkey installed.

1. **Download AHK v2:** Click here to download [AutoHotkey v2.0](https://www.autohotkey.com/download/ahk-v2.exe).
2. **Run as Admin:** Right-click the downloaded script file and select **"Run as Administrator"** (required for Roblox to register the keystrokes).

---

## 🎮 Script Guides

### 1. Auto-Buying Seeds
Use this script to completely automate your seed purchasing loop.

* **Positioning:** Stand directly behind the lever. Move your character as close to it as possible.
* **Camera Angle:** Look directly down at the floor.

### 2. Auto-Buying Eggs
Use this script to automatically buy eggs from the shelves.

* **Positioning:** Walk inside the buying shelf and align your character flat against the corner of either side.
* **Camera Angle:** Position your crosshair straight ahead, then tilt your view slightly downward until the first egg purchase prompt enters your sight.

---

## ⌨️ Controls & Hotkeys

Both scripts share the exact same simple control scheme:

| Hotkey | Action | Description |
| :---: | :--- | :--- |
| **`F1`** | **Start / Pause** | Toggles the automation on or off seamlessly. |
| **`F2`** | **Emergency Exit** | Instantly kills the script if a bug or misalignment occurs. |

---

> 💡 **Troubleshooting Tip:** If the script is running but your character isn't moving or clicking in-game, close the script, right-click the file, and choose **Run as Administrator**. Roblox's anti-cheat sometimes blocks virtual inputs from non-admin apps.
