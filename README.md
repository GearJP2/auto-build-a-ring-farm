This repository contains AutoHotkey (AHK) scripts designed to automate two specific tasks in Roblox's Build A Ring:

Auto-Buying Seeds

Auto-Buying Eggs

Each script is contained in its own separate file for ease of use.

📄 Instructions
1. Auto-Buying Seeds
To use the seed-buying script, set up your character and camera as follows:

Positioning: Stand directly behind the lever, keeping your character as close to it as possible.

Camera Angle: Adjust your camera so you are looking straight down at the floor.

Controls: * Press F1 to Start / Pause the script.

Press F2 to instantly Exit the script if it bugs out.

2. Auto-Buying Eggs
To use the egg-buying script, set up your character and camera as follows:

Positioning: Walk inside the buying shelf and align your character flat against the corner of either side.

Camera Angle: Position your crosshair straight ahead, then tilt your camera slightly downward until the first egg purchase prompt is in your field of view.

Controls: * Press F1 to Start / Pause the script.

Press F2 to instantly Exit the script if it bugs out.

⚠️ Note: Ensure you have AutoHotkey installed on your PC before running these files. Run the scripts as Administrator if they do not register inputs inside the Roblox client.
Install your AHK -> https://www.autohotkey.com/download/ahk-v2.exe
